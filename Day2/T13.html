In my Multi-Agent Reinforcement Learning (MARL) Snake Game project, I implemented multiple autonomous snakes that learn to navigate the grid and avoid collisions using a shared Deep Q-Network (DQN). Each snake observes its environment and makes decisions based on learned policies. The environment is built in Python without using torch or CUDA, and the training occurs in Google Colab. To start the game manually in the local viewer, the user needs to press Ctrl + R to run the script and Esc to exit. The project showcases intelligent agent coordination and real-time movement in a visually distinct grid environment.
